## 2. AnimationBuilderDock

At the end of right side dock, should appear AnimationBuilderDock
It containts few params
- FilePath -> Path to json file
- LibName -> library name from file
- Sprite2D path -> path to sprite2D from scene root
- overwrite -> allows for overwriting library if library with same name exist

after setting all of params just press ``Bake animations`` button.

Animations should be added to animation player
